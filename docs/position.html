Позиция

Утилиты для управления расположением элемента в DOM.

Справочник классов по умолчанию

Класс
Свойства
staticposition: static;
fixedposition: fixed;
absoluteposition: absolute;
relativeposition: relative;
stickyposition: sticky;

Static

Используйте static для позиционирования элемента в соответствии с обычным потоком документа.

Любые смещения будут игнорироваться, и элемент не будет действовать как ссылка на позицию для абсолютно позиционированных дочерних элементов.

Статический родитель

Абсолютный ребенок

<div class="static ...">
  <p>Статический родитель</p>
  <div class="absolute bottom-0 left-0 ...">
    <p>Абсолютный ребенок</p>
  </div>
</div>

Relative

Используйте relative, чтобы позиционировать элемент в соответствии с нормальным потоком документа.

Смещения рассчитываются относительно нормального положения элемента, и элемент будет действовать как ссылка на позицию для абсолютно позиционированных дочерних элементов.

Статический родитель

Абсолютный ребенок

<div class="relative ...">
  <p>Статический родитель</p>
  <div class="absolute bottom-0 left-0 ...">
    <p>Абсолютный ребенок</p>
  </div>
</div>

Absolute

Используйте absolute для позиционирования элемента вне обычного потока документа, заставляя соседние элементы действовать так, как будто этот элемент не существует.

Смещения вычисляются относительно ближайшего родителя, у которого позиционирование, отличное от static, и элемент будет действовать как ссылка на позицию для других абсолютно позиционированных дочерних элементов.

Статический брат

<div class="static ...">
  <!-- Статический родитель -->
  <div class="static ..."><p>Статический ребенок</p></div>
  <div class="inline-block ..."><p>Статический брат</p></div>
  <!-- Статический родитель -->
  <div class="absolute ..."><p>Абсолютный ребенок</p></div>
  <div class="inline-block ..."><p>Статический брат</p></div>
</div>

Fixed

Используйте fixed для позиционирования элемента относительно окна браузера.

Смещения вычисляются относительно области просмотра, и элемент будет действовать как ссылка на позицию для абсолютно позиционированных дочерних элементов.

Адаптивность

Чтобы изменить расположение элемента только в определенной контрольной точке, добавьте префикс {screen}: к любой существующей утилите позиции. Например, добавление класса md:absolute к элементу применит утилиту absolute при средних размерах экрана и выше.

<div class="relative h-32 ...">
  <div class="inset-x-0 bottom-0 relative md:absolute"></div>
</div>

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией Адаптивного дизайна.

Кастомизация

Варианты

По умолчанию, только responsive варианты создаются для утилит position.

Вы можете контролировать, какие варианты создаются для утилит position для изменения свойства position в разделе variants Вашего файла конфигурации tailwind.config.js.

Например, эта конфигурация также будет генерировать варианты hover и focus:

  // tailwind.config.js
  module.exports = {
    variants: {
      extend: {
        // ...
+       position: ['hover', 'focus'],
      }
    }
  }

Отключение

Если Вы не планируете использовать в своем проекте утилиты для изменения position, Вы можете полностью отключить их, установив для position свойство значение false в разделе corePlugins Вашего файла конфигурации:

  // tailwind.config.js
  module.exports = {
    corePlugins: {
      // ...
+     position: false,
    }
  }
